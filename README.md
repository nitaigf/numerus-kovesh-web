# Numerus Kovesh Web

Frontend SolidJS do ecossistema Numerus Kovesh.

## Stack

- SolidJS
- Vite
- TypeScript
- Bun
- Vitest

## Executar localmente

```bash
bun install
cp .env.example .env
bun run dev
```

Frontend local: http://localhost:5173

## Variaveis de ambiente

```env
VITE_API_BASE_URL=http://localhost:8010
```

## Testes

```bash
bun run test
```

## Build

```bash
bun run build
```

## Deploy

Deploy recomendado na Vercel.
